# Word Racing 自适应学习系统设计

> **版本**: v1.0
> **设计目标**: 让孩子每天15分钟背单词 + 5分钟玩赛车，形成持续学习习惯
> **设计日期**: 2026-05-24

---

## 一、现状问题分析

| 问题 | 现状 | 影响 |
|------|------|------|
| 出题无记忆 | 随机抽取，已掌握的词反复出现 | 浪费时间，效率低 |
| 难度不递进 | 只有筛选 `level <= maxLevel`，无个性化 | 无法匹配孩子实际水平 |
| 无进度感知 | 不知道刷了多少词、掌握多少 | 缺乏成就感，难以坚持 |
| 每套题量少 | 固定5题 | 频繁切界面，打断学习节奏 |
| 赛车比例失衡 | 答题后直接玩赛车 | 孩子会"混"过答题去玩赛车 |

---

## 二、核心设计理念

### 2.1 学习闭环

```
评估 → 学习 → 巩固 → 奖励 → 评估...
```

1. **评估**：首次出现的词 → 标记为"新词"，答对/答错记录
2. **学习**：错词进入"待巩固"队列，优先出题
3. **巩固**：间隔复习（艾宾浩斯遗忘曲线）
4. **奖励**：解锁赛车资格、成就徽章、每日目标进度

### 2.2 词库分层

| 状态 | 定义 | 出题权重 | 复习周期 |
|------|------|----------|----------|
| **未学习** | 从未出现过 | 3x（优先） | - |
| **学习中** | 答错过，未达到掌握标准 | 2x | 每套必出 |
| **巩固中** | 连续答对1次 | 1x | 1天后复习 |
| **已掌握** | 连续答对3次，间隔>3天 | 0x（不出现） | 7天后抽查 |
| **生疏** | 已掌握后再次答错 | 2x | 降级为"学习中" |

### 2.3 每日学习模型

```
每日目标 = 学新词 N 个 + 复习错词 M 个

推荐配置：
- 新手期（前7天）：每天5个新词 + 复习所有错词
- 稳定期：每天10个新词 + 复习错词 + 巩固词
- 进阶期：每天15个新词 + 复习 + 抽查已掌握词
```

## 四、出题算法

### 4.1 智能选题策略

```javascript
/**
 * 每套题的构成：
 * - 40% 新词/学习中词汇（优先未学习）
 * - 30% 待复习词汇（错词、巩固期词汇）
 * - 20% 巩固中词汇（间隔到期）
 * - 10% 随机词汇（保持多样性）
 */

function selectWordsForQuiz(allWords, progress, config) {
  const today = new Date().toDateString();

  // 1. 获取各类型词汇
  const unlearned = allWords.filter(w => !progress[w.word]);
  const learning = allWords.filter(w =>
    progress[w.word]?.status === 'learning'
  );
  const dueForReview = allWords.filter(w => {
    const p = progress[w.word];
    if (!p) return false;
    // 错词优先
    if (p.status === 'learning' && p.wrongCount > 0) return true;
    // 巩固期到期
    if (p.status === 'consolidating' && new Date(p.nextReviewDate) <= new Date()) return true;
    return false;
  });
  const consolidating = allWords.filter(w =>
    progress[w.word]?.status === 'consolidating'
  );

  // 2. 按比例选题
  const selected = [];
  const quizSize = config.quizSize || 10;

  // 40% 新词/学习中
  const newCount = Math.ceil(quizSize * 0.4);
  const newWords = shuffle([...unlearned, ...learning]).slice(0, newCount);
  selected.push(...newWords);

  // 30% 待复习
  const reviewCount = Math.ceil(quizSize * 0.3);
  const reviewWords = shuffle(dueForReview).slice(0, reviewCount);
  selected.push(...reviewWords);

  // 20% 巩固中
  const consolidatingCount = Math.ceil(quizSize * 0.2);
  const consolidatingWords = shuffle(consolidating).slice(0, consolidatingCount);
  selected.push(...consolidatingWords);

  // 10% 随机（填充到目标数量）
  const remaining = quizSize - selected.length;
  if (remaining > 0) {
    const remainingPool = allWords.filter(w =>
      !selected.includes(w) &&
      progress[w.word]?.status !== 'mastered'
    );
    selected.push(...shuffle(remainingPool).slice(0, remaining));
  }

  return shuffle(selected);
}
```

### 4.2 难度递进

```javascript
/**
 * 难度自适应：
 * - 根据当前正确率动态调整
 * - 连续3套正确率>90% → 提升难度
 * - 连续2套正确率<60% → 降低难度
 */

function adjustDifficulty(recentAccuracy) {
  // recentAccuracy = 最近5套题的正确率数组

  const recent = recentAccuracy.slice(-5);
  const avg = recent.reduce((a, b) => a + b, 0) / recent.length;

  if (avg > 0.9 && recent.filter(r => r > 0.9).length >= 3) {
    // 提升难度
    return { action: 'increase', amount: 1 };
  } else if (avg < 0.6 && recent.filter(r => r < 0.6).length >= 2) {
    // 降低难度
    return { action: 'decrease', amount: 1 };
  }

  return { action: 'maintain' };
}
```

## 五、赛车解锁机制

### 5.1 学习解锁赛车

**核心原则**：赛车是学习的奖励，不是逃避学习的途径

```
解锁条件（满足任意一项）：
1. 今日学习目标完成（新词达标 + 错词复习完）
2. 本套题正确率 ≥ 70% 且 答题数 ≥ 8题

赛车时间限制：
- 每次赛车消耗 1 张"赛车券"
- 赛车券获取方式：
  - 完成1套10题 → 获得1张
  - 正确率≥90% → 额外获得1张
  - 连击5次以上 → 额外获得1张
- 每日最多积累5张赛车券
```

### 5.2 学习-赛车时间比例

```
目标：学习15分钟 : 赛车5分钟 = 3:1

实现方式：
- 答题时间计入"学习时间"
- 赛车时间计入"游戏时间"
- 每日结束时显示时间分布报告
- 提示："今天学习XX分钟，赛车XX分钟，学习占比XX%"
```

## 七、复习机制（艾宾浩斯）

### 7.1 复习间隔

```javascript
const reviewIntervals = {
  learning: 0,           // 立即复习（下一套题）
  consolidating: [       // 巩固期复习间隔
    1 * 24 * 60 * 60 * 1000,  // 1天后
    3 * 24 * 60 * 60 * 1000,  // 3天后
    7 * 24 * 60 * 60 * 1000,  // 7天后
    14 * 24 * 60 * 60 * 1000, // 14天后
  ],
  mastered: 30 * 24 * 60 * 60 * 1000, // 30天后抽查
};
```

### 7.2 复习状态转换

```
新词 ─答对→ 巩固中(1天后复习)
    └答错→ 学习中(立即复习)

学习中 ─答对→ 巩固中
       └答错→ 保持学习中

巩固中 ─答对→ 下次复习间隔延长
       └答错→ 学习中

已掌握 ─抽查答对→ 保持已掌握
       └抽查答错→ 生疏 → 学习中
```

## 八、题量与时间规划

### 8.1 每套题量调整

| 场景 | 题量 | 预计时间 | 说明 |
|------|------|----------|------|
| 快速复习 | 5题 | 2-3分钟 | 仅复习错词 |
| 标准学习 | 10题 | 5-8分钟 | 日常学习 |
| 深度学习 | 15题 | 8-12分钟 | 周末/假期 |
| 闪电挑战 | 5题 | 2分钟 | 全对奖励翻倍 |

### 8.2 每日15分钟学习规划

```
建议学习流程：

1. 打开游戏 → 显示今日目标和学习进度 (30秒)
2. 第一套标准学习 (10题, 6分钟)
3. 查看结果 → 错词学习 (1分钟)
4. 获得赛车券 → 玩一次赛车 (2分钟)
5. 第二套快速复习 (5题, 3分钟)
6. 查看今日成就 (30秒)
7. 继续学习或结束

总时间：约15分钟学习 + 2-3分钟赛车
```

## 十、风险与应对

| 风险 | 应对措施 |
|------|----------|
| 孩子为了玩赛车故意答错快进 | 赛车需要正确率门槛，故意答错无法获得赛车券 |
| 词库太小，快速学完 | 支持动态扩展词库，已掌握词30天后抽查 |
| 学习曲线太陡 | 自适应难度，连续表现差自动降级 |
| 孩子不喜欢复习模式 | 复习题目用不同题型（填空、音标等）增加趣味性 |
| 家长无法监控 | 添加家长面板，显示学习报告 |
