# 词库难度分类示例（30词）

## 分类规则

```
评分维度：
1. 词长（越短越简单）
   - ≤4字母: +0分
   - 5-6字母: +1分
   - 7-8字母: +2分
   - ≥9字母: +3分

2. 音节（越少越简单）
   - 1音节: +0分
   - 2音节: +1分
   - 3音节: +2分
   - ≥4音节: +3分

3. 词频/常见度（主观判断）
   - 小学已学: +0分
   - 初中常见: +1分
   - 初中进阶: +2分
   - 高中词汇: +3分

4. 多义词/语法词
   - 单义实词: +0分
   - 多义词: +1分
   - 语法功能词: +1分

总分 → Level：
- 0-1分: Level 1（基础入门）
- 2-3分: Level 2（核心词汇）
- 4-5分: Level 3（进阶词汇）
- 6-7分: Level 4（挑战词汇）
- ≥8分: Level 5（拓展词汇）
```

## 分类反思

### 可能存在的问题

1. **Level 1 vs Level 2 边界模糊**
   - `finger`, `space` 是否应该是 Level 1？
   - 判断依据：是否小学低年级已学

2. **合成词处理**
   - `playground` (10字母) vs `everywhere` (10字母)
   - 前者 Level 2（小学已学），后者 Level 3（初中语法）
   - 单纯看词长会误判，需要人工标注

3. **派生词**
   - `unhappy`, `careless`, `recently` 等
   - 如果认识词根，难度降低
   - 可能需要单独处理常见词缀

### 建议调整

```
方案A：纯算法分类
  - 优点：可批量处理1982词
  - 缺点：边界情况不准确

方案B：算法初分 + 人工校验
  - 先用算法分出大致等级
  - 重点校验边界词（总分在临界值的词）

方案C：基于教材单元标注
  - 查六年级上下册目录，按单元标注
  - 最准确但工作量大
```
