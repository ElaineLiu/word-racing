## 二、新增需求分析

### 2.1 自适应学习系统 v2 核心需求

1. **单词掌握状态追踪**
   - 每个词需要记录：状态、简单题/复杂题是否通过、答错次数、首次/最后/掌握日期

2. **每日学习流程**
   - 固定每天3套题，每套10题
   - 追踪今日进度：套数、新学词数、复习词数
   - 今日金币累积结算

3. **自适应出题算法**
   - 错词优先复习（最多3题）
   - 简单题通过→出复杂题检查
   - 复杂题通过→出简单题检查
   - 新词填充

4. **答题会话管理**
   - 防止中途退出丢失进度
   - 支持断点续答

5. **金币系统升级**
   - 简单题答对：燃油币
   - 复杂题答对：装备币
   - 连击奖励
   - 每日目标奖励

## 五、变量分类：持久化 vs 运行时

### 5.1 持久化存储变量（localStorage）

| 存储键 | 变量 | 用途 | 更新时机 |
|--------|------|------|----------|
| `wr_game_state` | 游戏核心状态 | 资源、升级、统计 | 每次状态变更 |
| `wr_word_progress` | 单词掌握状态 | 学习进度追踪 | 每次答题后 |
| `wr_daily_stats` | 每日统计 | 历史记录、报表 | 每日结算 |
| `wr_quiz_session` | 答题会话 | 断点续答 | 每题答题后 |
| `wr_wordset_config` | 词库选择 | 用户偏好 | 切换词库时 |

### 5.2 运行时变量（内存）

| 模块 | 变量 | 用途 | 生命周期 |
|------|------|------|----------|
| `ProgressTracker` | `wordProgress` | 单词进度缓存 | 页面会话 |
| `DailyManager` | `todayStats` | 今日统计缓存 | 页面会话 |
| `QuizSessionManager` | `session` | 当前答题会话 | 答题流程 |
| `AdaptiveSelector` | `candidatePool` | 候选词池 | 每次选题 |
| `QuizEngine` | `currentQuiz` | 当前题目 | 每套题 |
| `VocabularyQuiz` | `combo` | 连击计数 | 每套题 |

### 5.3 协同响应变量（事件驱动）

| 事件 | 响应者 | 响应动作 |
|------|--------|----------|
| `QUIZ_ANSWER` | ProgressTracker | 更新单词状态 |
| `QUIZ_ANSWER` | QuizSessionManager | 记录答题、保存进度 |
| `QUIZ_ANSWER` | DailyManager | 更新今日统计 |
| `QUIZ_COMPLETE` | GameState | 结算金币、更新统计 |
| `QUIZ_COMPLETE` | QuizSessionManager | 保存套题结果 |
| `QUIZ_COMPLETE` | DailyManager | 检查每日目标 |
| `DAILY_GOAL_COMPLETE` | GameState | 发放目标奖励 |
| `WORD_MASTERED` | ProgressTracker | 更新掌握状态 |
| `SESSION_RESUME` | QuizSessionManager | 恢复答题进度 |

## 八、接口设计

### 8.1 ProgressTracker（单词进度追踪）

```javascript
// learning/progress-tracker.js

export class ProgressTracker {
  constructor(eventBus, wordSetId);

  // 获取单词状态
  getStatus(wordText: string): MasteryStatus;

  // 更新单词状态（答题后调用）
  updateStatus(wordText: string, mode: string, correct: boolean): void;

  // 批量获取状态
  getStatuses(wordTexts: string[]): Map<string, MasteryStatus>;

  // 获取待复习词（错词）
  getWrongWords(): WordProgress[];

  // 获取待检查词（单题型通过）
  getCheckWords(): { needSimpleCheck: WordProgress[], needComplexCheck: WordProgress[] };

  // 获取已掌握词
  getMasteredWords(): WordProgress[];

  // 获取统计
  getStats(): { total, mastered, learning, unlearned };

  // 持久化
  save(): void;
  load(): void;
}
```
